# 70-01 Отчёт по кнопке за период (7/14/30)

## Business goal
Пользователь по кнопке получает отчёт за выбранный период: тренировки, вес, прогресс-фото/замеры, питание.

## Scope
- Кнопка «Отчёт» → выбор периода 7/14/30.
- Сводка:
  - тренировки: кол-во, по ключевым упражнениям (упрощённо: подходы/объём)
  - вес: первое/последнее значение и разница
  - замеры: наличие и последние значения
  - фото: кол-во и последние даты
  - калории: сумма/среднее по логу

## Acceptance criteria
- Отчёт формируется быстро и без ошибок при отсутствии части данных.
- Формат отчёта читаемый (1–2 сообщения максимум).

## Tests
- Пустые данные; частичные данные; полный набор.
