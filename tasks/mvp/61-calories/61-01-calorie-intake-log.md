# 61-01 Учёт потреблённых калорий (лог)

## Business goal
Фиксировать фактически потреблённые калории (не только рекомендованное меню).

## Scope
- Добавить запись: калории + (опц) комментарий + дата/время.
- История: 7/14/30.

## UX flow
- «Питание» → «Калории (добавить)» → ввод числа → (комментарий опц.)
- «Калории (история)» → период → список + суммарно/среднее.

## Acceptance criteria
- Добавление записи ≤2 шага.
- История показывает суммарно и среднее за период.

## Future
- Определение калорий по фото.

## Tests
- Добавление/история/агрегация.
