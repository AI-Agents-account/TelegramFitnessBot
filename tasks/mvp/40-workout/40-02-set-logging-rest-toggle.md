# 40-02 Логирование подходов + кнопка отдыха (toggle)

## Business goal
Во время тренировки пользователь выбирает упражнение и быстро добавляет подход: вес + повторы. Отдых фиксируется одной кнопкой «Начать отдых / Завершить отдых».

## Scope
- `WorkoutSet`:
  - `exerciseId`, `setIndex`, `weight`, `reps`
  - `restStartedAt`, `restEndedAt` (опц.)
- UX:
  - «Ввести подход» → запрос веса → запрос повторов → подтверждение
  - toggle-кнопка отдыха:
    - если отдыха нет → установить `restStartedAt=now`
    - если отдых идет → установить `restEndedAt=now`

## Acceptance criteria
- Добавление подхода занимает 2 коротких числовых ввода.
- Toggle отдыха меняет подпись и состояние.
- История подходов по упражнению доступна в текущей сессии.

## Technical notes
- Уточнить правило: отдых привязываем к последнему подходу или к упражнению/сессии.
  - MVP рекомендация: хранить пары restStart/restEnd на уровне сессии (таблица events) или у последнего сетта.

## Tests
- Добавление 3 подходов; отдых start/stop; корректные timestamps.
