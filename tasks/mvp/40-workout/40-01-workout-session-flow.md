# 40-01 Проведение тренировки: старт/продолжить/завершить

## Business goal
Бот должен быть удобен «на тренировке»: стартовать сессию по выбранному дню плана, продолжать незавершённую, завершать с фиксацией результата.

## Scope
- `WorkoutSession` со статусом IN_PROGRESS/COMPLETED.
- Старт:
  - выбор дня плана
  - создание сессии
- Продолжить:
  - если есть IN_PROGRESS — открыть текущий экран упражнения
- Завершить:
  - закрыть сессию, зафиксировать `endedAt`

## UX flow
- «Тренировка» → «Начать» / «Продолжить»
- Внутри: «Следующее упражнение», «Предыдущее», «Завершить тренировку»

## Acceptance criteria
- У пользователя не бывает двух активных сессий одновременно.
- После завершения — краткое резюме (упражнения + кол-во подходов).

## Technical notes
- Хранить текущий выбранный `exerciseId`/позицию в сессии.

## Smoke test
- Начать → выйти → продолжить → завершить.
