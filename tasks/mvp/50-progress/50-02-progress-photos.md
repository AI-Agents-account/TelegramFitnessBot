# 50-02 Прогресс: фото

## Business goal
Привязка фото прогресса к дате, просмотр истории.

## Scope
- Пользователь отправляет фото → бот сохраняет Telegram `file_id` + дату.
- (Опционально) выбор типа: front/side/back.
- История: показать последние N фото/дат.

## Acceptance criteria
- Фото сохраняется и можно запросить «последние фото».
- Пользователь может удалить фото (минимум — в день загрузки).

## Technical notes
- Хранить `file_id` достаточно для пересылки/повторного показа.

## Tests
- Загрузка, просмотр, удаление.
