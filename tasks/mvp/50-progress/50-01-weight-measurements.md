# 50-01 Прогресс: вес и обхваты

## Business goal
Быстро фиксировать прогресс тела: вес и обхваты, смотреть историю.

## Scope
- Ввод веса (отдельно) и/или вместе с обхватами.
- Фиксированный набор полей (MVP): талия, грудь, бедра (можно расширять позже).
- История: 7/14/30 дней.

## UX flow
- «Прогресс» → «Новый замер» → (вес) → (обхваты) → сохранить
- «История» → период → текстовая сводка

## Acceptance criteria
- Можно пропускать часть обхватов.
- История корректно ограничена периодом.

## Tests
- Ввод/история, пропуски полей.
